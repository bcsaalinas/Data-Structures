# Python Data Structures & Algorithms

A collection of data structure implementations, algorithms, and exercises in Python.

## Repository Structure

- `data_structures/`: Core implementations of data structures (Arrays, Stacks, Queues, HashMaps, Trees).
- `algorithms/`: Common algorithms including Sorting and Math.
- `exercises/`: Solved problems from exams, contests, and general practice.
- `projects/`: Larger implementations like the Spell Checker.

## How to use
Most files are self-contained Python scripts or Jupyter Notebooks. You can run them directly using `python3` or a Jupyter environment.
